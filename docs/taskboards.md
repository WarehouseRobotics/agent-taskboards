# What Does The Agent Taskboards App Do

## Technological Pillars

UI: 

* React, tailwind.css

Backend:

* node-llama-cpp for text embeddings models
* sqlite and sqlite-vec for storage and vector search


## Key Advantages

* privacy: local-first text embeddings, runs in Docker locally
* no-frills: fast, low memory footprint (but single-user)
* acts as management infrastructure and doubles as vector-search memory backend for coding agents
